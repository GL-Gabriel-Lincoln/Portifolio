Exemplo: Mostre que P² é par, p E Z, então p é par.

Solução:
Primeiro mostremos que o produto de dois números ímpares é ímpar

Sejam x e y números ímpares e portanto existem m, n E Z tais que

x = 2n+1 e y = 2.m+1

assim x . y = (2n + 1) . (2m + 1)

            = 4n.m + 2n + 2m + 1

            = 2.(2nm + n + m) + 1   -  ax + ab = a .(x + b)

e portando x . y é um número ímpar.

Agora se p² é par, p não pode ser ímpar, uma vez que se p fosse ímpar, P.P = P² seria ímpar, o que não ocorre por hipótese.

Exemplo: Mostre que sqrt(2) não é racional

Suponha que existam p, q E Z tais que

sqrt(2) = p/q , com p/q na forma irredutivel.

então 2 = p²/q²

p² = 2q²

Dessa forma, segue que p² é par e pelo exemplo anterior p é par

sendo p par, existe K E Z tal que 
p = 2K

Assim 

(2k)² = 2q² => 4k² = 2q² => 2k² = q²

Portanto q² é par e consequentemente q é par

Assim p, q são inteiros pares e portanto p/q não está na forma irredutivel, o que é uma Contradição

Para o proposito de Geometria de Analítica é suficiente o seguinte resultado:

A cada ponto da reta R corresponde um único número (racional ou irracional)

Valor Absoluto:

Se X é um numero real, o módulo de x (ou valor absoluto) de x é o número |x| definido por :

|x| = { x se x >= 0} 
      {-x se x < 0}

Exemplo:

|-5| = -(-5) => |-5| = 5

|2| = 2

|-150| = -(-150) = |-150| = 150

Interpretação Geometrica:
